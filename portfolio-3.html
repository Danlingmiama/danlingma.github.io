NLP - Restaurant Review: Aspect Based Sentiment Analysis

Overview

  • Fall, 2019

This Aspect-based Sentiment Analysis (ABSA) is based on two machine learning models: The Aspect Categories Model and the Sentiment Model. The Aspect Categories Model is used to predict sentence category based on features generated by aspect terms. The Sentiment Model is used to predict the sentiment of the sentence based on features generated by polarity terms.
